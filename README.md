# PrecisionGamingOptimizer (Balanced)

Balanced Windows + game profiles for smoother FPS and lower latency on **low-end to mid-range** PCs.

- Smart hardware auto-detect (CPU/RAM/GPU/Tier)
- Balanced base profile (Windows/CPU/RAM/Network)
- Game profiles:
  - FiveM
  - Fortnite
- One-click revert
