# Benchmark Guide (Before / After)

Use consistent settings and repeat tests.

## 1) Prepare
- Close launchers you don't need
- Reboot before "Before" benchmark
- Use the same graphics settings and resolution

## 2) Measure
### FPS / Frame-time
- MSI Afterburner + RivaTuner (FPS + 1% low)
- CapFrameX (recommended for frame-time analysis)

### Latency
- In-game latency meter (Fortnite)
- Network ping (same server region)

## 3) Record
Create a simple table:

| Game | Scenario | Avg FPS | 1% Low | Stutter notes | Ping |
|------|----------|---------|--------|---------------|------|

## 4) Compare
- Focus on **1% lows** and **stutters**, not just max FPS.
